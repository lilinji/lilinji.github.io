## 风格规范 (pixel-art)

### 配色方案

| 角色   | 颜色     | Hex     |
| ------ | -------- | ------- |
| 背景   | 淡蓝色   | #87CEEB |
| 主元素 | 深海军蓝 | #1A1A2E |
| 强调1  | 像素绿   | #00FF00 |
| 强调2  | 像素红   | #FF0000 |
| 强调3  | 像素黄   | #FFFF00 |
| 强调4  | 像素青   | #00FFFF |

### 设计规则

- 所有元素使用可见的像素结构
- 限制 16-32 色调色板
- 使用粗块状几何形状
- 添加怀旧游戏元素（星星、爱心、箭头）
- 禁止平滑渐变或抗锯齿
