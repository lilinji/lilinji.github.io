插图主题: 主管模式 Supervisor Pattern
风格: pixel-art (8-bit 像素画)

视觉构图:

- 层级结构图，类似游戏中的队伍选择界面
- 顶部中央：一个戴皇冠的大型 Boss 像素角色（Manager），穿蓝色战袍，体型比下属大1.5倍
- 下方三个小型像素角色呈扇形排列：
  - 左边：红色工人角色（牛马Agent），拿着锤子
  - 中间：绿色学者角色（研究Agent），戴眼镜拿书
  - 右边：黄色盾牌角色（安全Agent），拿盾牌
- 用像素风格的闪电箭头从 Boss 指向各个工人
- 底部汇聚点显示一个像素宝箱（综合结果）

配色方案:

- 背景: 深紫色渐变 #1A1A2E 到 #2D1B4E
- Boss角色: 金色边框 #FFD700，蓝色主体
- 工人角色: 各自鲜明的 8-bit 原色

文字内容:

- Boss 上方标注 "Manager"
- 底部标注 "Supervisor Pattern"

风格要点: 保持像素网格一致性，游戏风格的 UI 边框装饰，8-bit 字体
