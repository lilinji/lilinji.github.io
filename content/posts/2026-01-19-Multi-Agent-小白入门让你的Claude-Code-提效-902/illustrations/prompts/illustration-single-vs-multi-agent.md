插图主题: 单Agent vs 多Agent 对比
风格: pixel-art (8-bit 像素画)

视觉构图:

- 画面分为左右两部分对比
- 左侧：一个疲惫的蓝色像素小人，独自搬运一个巨大的红色箱子，头顶有汗滴和疲惫符号，进度条只有20%
- 右侧：三个不同颜色（绿、橙、紫）的像素小人一起协作搬运同样的箱子，轻松愉快，头顶有星星和笑脸，进度条100%
- 中间用像素风格的 VS 符号分隔

配色方案:

- 背景: 淡蓝色 #87CEEB
- 像素角色: 使用经典 8-bit 游戏色彩（绿 #00FF00, 红 #FF0000, 黄 #FFFF00）
- 进度条: 左侧红色（失败感），右侧绿色（成功感）

文字内容:

- 左侧标注 "Single Agent"
- 右侧标注 "Multi-Agent"

风格要点: 保持像素网格一致性，使用16-32色限制调色板，粗块状几何形状，不使用平滑渐变
