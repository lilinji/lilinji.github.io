插图主题: 流水线模式 Pipeline Pattern
风格: pixel-art (8-bit 像素画)

视觉构图:

- 横向工厂流水线场景
- 四个像素角色站在传送带旁边，从左到右：
  1. 产品经理：紫色，拿着剪贴板，输出"需求文档"图标
  2. 架构师：蓝色，戴眼镜，输出"设计图纸"图标
  3. 程序员：绿色，戴耳机，输出"代码文件"图标
  4. 审查员：橙色，拿放大镜，输出"通过勋章"图标
- 传送带用像素点组成的移动带表示，有箭头指向右侧
- 每个角色之间有一个小型的像素文件在传送带上移动
- 类似超级玛丽工厂关卡的视觉风格

配色方案:

- 背景: 淡蓝色 #87CEEB，有像素云朵
- 传送带: 灰色金属质感 #808080
- 文档图标: 白底黑字的像素风格

文字内容:

- 顶部标注 "Pipeline Pattern"
- 每个角色下方标注其角色名

风格要点: 保持像素网格一致性，工厂/游戏场景风格，传送带动态感
