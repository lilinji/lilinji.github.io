# 文章自动配图工作流

## 使用方法

复制以下 prompt 并替换 `[文章路径]` 和 `[风格]`：

```
为这篇文章自动配图，一键完成全部流程，每个章节都要有配图：
文章: [文章路径]
风格: [风格]

要求：
1. 分析文章结构，识别所有章节
2. 为每个章节生成对应插图
3. 复制图片到文章的 imgs 目录
4. 更新文章，插入所有图片引用
5. 确保覆盖率 100%
```

## 可用风格

| 风格    | 描述                             | 适用场景          |
| ------- | -------------------------------- | ----------------- |
| tech    | 深色科技感、霓虹蓝绿、仪表板风格 | 技术文章、AI/编程 |
| sketch  | 手绘铅笔、笔记本风格、灰蓝配色   | 学习笔记、教程    |
| elegant | 渐变紫粉、优雅简约               | 设计类、产品介绍  |
| minimal | 极简黑白线条                     | 严肃学术、商业    |
| warm    | 暖橙色调、亲和力                 | 生活类、故事      |
| playful | 多彩活泼、卡通风                 | 创意类、儿童向    |
| nature  | 自然绿色、有机感                 | 环保、健康        |
| bold    | 高对比、大胆配色                 | 营销、宣传        |
| notion  | Notion 黑白图标风                | 工具类、效率      |
